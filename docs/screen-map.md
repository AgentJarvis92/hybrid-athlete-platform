# MVP Screen Map — Hybrid Athlete Training Platform

**Updated:** 2026-03-05

---

## Athlete Mobile App

### Screen 1 — Today Dashboard
**Purpose:** Show the athlete everything they need for the day.

Sections:
- Daily briefing (Training Brain)
- Run workout card
- Strength workout card
- Nutrition targets
- Recovery summary

Primary actions:
- Start workout
- Log meal
- View coach notes

---

### Screen 2 — Training Calendar
**Purpose:** Display the athlete's weekly training schedule.

Features:
- Weekly calendar view
- Workout cards for each day
- Tap workout to open details

---

### Screen 3 — Workout Details
**Purpose:** Display instructions for the selected workout.

Sections:
- Workout title
- Coach notes
- Exercises or run instructions

Button: Start Workout

---

### Screen 4 — Workout Execution
**Purpose:** Allow athletes to log strength exercises in real time.

Features:
- Exercise cards
- Set logging
- Weight input
- Rest timer
- Previous performance shown

---

### Screen 5 — Nutrition Logger
**Purpose:** Log meals and track macros.

Features:
- Food search
- Barcode scan
- Manual entry

Displays:
- Daily calories
- Protein
- Carbs
- Fat vs. daily targets

---

### Screen 6 — Progress Dashboard
**Purpose:** Show training trends over time.

Metrics:
- Weekly running mileage
- Strength progression
- Nutrition adherence
- Workout completion rate

---

### Screen 7 — Messages
**Purpose:** Communication between athlete and coach.

Features:
- Direct messages
- Workout comments

---

### Screen 8 — Settings
**Purpose:** Configure modules and account.

Options:
- Enable/disable modules (Running / Strength / Nutrition)
- Edit training goals
- Manage integrations (Strava, Apple Health)

---

## Coach Web Portal

### Screen 1 — Coach Dashboard
**Purpose:** Overview of all athletes.

Metrics:
- Workout completion
- Training status
- Recent activity

---

### Screen 2 — Athlete List
**Purpose:** List of athletes managed by the coach.

---

### Screen 3 — Athlete Profile
**Purpose:** Individual athlete data.

Metrics:
- Training completion
- Weekly mileage
- Strength workouts
- Nutrition adherence

---

### Screen 4 — Program Builder
**Purpose:** Create weekly training programs.

Features:
- Assign workouts to days
- Edit sessions

---

### Screen 5 — Workout Builder
**Purpose:** Create individual workouts.

Sections:
- Warm-Up
- Main Work
- Accessory Work
- Cool-Down

---

### Screen 6 — Messaging
**Purpose:** Coach communicates with athletes.

Features:
- Direct messaging
- Workout feedback
