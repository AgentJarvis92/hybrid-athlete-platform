# Hybrid Athlete Training Platform — Product Vision

**Updated:** 2026-03-05

---

## Vision

Build the training operating system for modern athletes.

Today athletes use multiple disconnected apps (Runna, Hevy, Cronometer, Strava) that don't communicate. A long run doesn't update nutrition targets. Strength workouts don't affect running intensity. Coaches can't see everything in one place.

This platform is the central brain for training, recovery, and nutrition — one unified system that understands the athlete's entire training life.

---

## Target Users

**Primary:** Hybrid athletes combining endurance + strength + structured nutrition
- Marathon runners who lift
- HYROX athletes
- Triathletes
- Endurance athletes who cross-train

**Secondary:** Coaches managing hybrid athletes

---

## Core Product Principle

Every recommendation considers: running workload + strength workload + nutrition intake + recovery indicators.

Training is never managed in isolation.

---

## Modular System

Athletes choose modules during onboarding. Interface adapts dynamically.

| Use Case | Modules |
|----------|---------|
| Runner | Running + Nutrition |
| Lifter | Strength + Nutrition |
| Hybrid Athlete | Running + Strength + Nutrition |

Modules: Running / Strength Training / Nutrition Tracking / Recovery Monitoring

---

## Coaching Model

Two systems: **AI Coach** and **Human Coach**. Athletes may use either or both.

If both are active, human coach overrides AI adjustments.

---

## Competitors

| App | Strength | Weakness |
|-----|----------|----------|
| Runna | Great run plans | No strength, no nutrition |
| Hevy | Clean lifting UI | No running, no nutrition, no coach portal |
| Cronometer | Best micronutrient tracking | Manual entry, no training context |
| Cal AI | Fast photo logging | Just calories, no training integration |
| TrueCoach | Good trainer portal | No running plans, no nutrition |
| TrainingPeaks | Pro-level periodization | Complex, expensive, no nutrition |

**Moat:** Only platform connecting all three domains with cross-domain AI intelligence.
